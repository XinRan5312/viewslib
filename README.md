# viewslib
UI View  更具自己工作中遇到的问题慢慢积攒实现的如下的一个综合的工具类
支持SrollView 嵌套多了listview下拉刷新  上拉加载更多。
自定义那个HotTab和底部Tad动态加载
里面还有一个FragmentStatePagerAdapter的支持
支持RecylerView下拉刷洗和上拉加载更多。
